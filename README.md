# Hello World

A project that demonstrates how to create a Python package.
The code in it just prints "Hello World".

## Installation

Run the following to install:
```python
pip install hello
```

## Usage

```python
from hello import say_hello

# Generate "Hello World"
say_hello()

# Generate "Hello Everybody"
say_hello("Everybody")
```
